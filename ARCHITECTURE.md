## Database Schema Details

```sql
CREATE TABLE students (
  id SERIAL PRIMARY KEY,
  name VARCHAR(255) NOT NULL,
  email VARCHAR(255) UNIQUE NOT NULL,
  password VARCHAR(255) NOT NULL,
  otp VARCHAR(6),
  otp_expiry TIMESTAMP,
  is_verified BOOLEAN DEFAULT FALSE,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  updated_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- Indexes for performance
CREATE INDEX idx_email ON students(email);
CREATE INDEX idx_created_at ON students(created_at);
```

**Field Explanations:**
- `id` - Unique identifier (auto-incremented)
- `name` - User's full name
- `email` - Unique email (indexed for fast lookup)
- `password` - Hashed password (never plain text)
- `otp` - Current OTP (6 digits), NULL if verified
- `otp_expiry` - Timestamp when OTP expires
- `is_verified` - Boolean flag for email verification status
- `created_at` - Account creation timestamp
- `updated_at` - Last modification timestamp
